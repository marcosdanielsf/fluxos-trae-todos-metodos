# Evidências — Fase 1 (Preparação & Diagnóstico 360)

Use as pastas 01…08 para anexar documentos/links por subtarefa. Padrões:
- Formatos aceitos: PDF, DOCX, XLSX/CSV, imagens, links internos
- Nomeação: `F1-<subtarefa>-<conteudo>-v<semana|data>.ext`
- Versão: mantenha changelog simples no README de cada pasta

Checklist rápido (por subtarefa)
1) Dados/Acessos/Stakeholders — acessos, mapa stakeholders, riscos
2) ICP/Personas/Segmentação — ICP, personas, critérios de segmentação
3) Competitiva/Benchmark — matrizes, quadro de posicionamento
4) Oferta/Produto/UX — mapas, evidências de auditoria, recomendações
5) Canais — funil, CRM, métricas e gargalos
6) Pesquisa — roteiros, amostra, resultados e síntese
7) CS/NPS/Churn — métricas, causa raiz e ações
8) Hipóteses/Prioridades/Plano — hipóteses, matriz, cronograma
